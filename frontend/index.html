About Me

Welcome to my portfolio! My name is Ethan Passino, and I’m a passionate software engineer with a love for learning and creating impactful projects. My journey in computer science began in high school, and it’s been a rewarding adventure ever since.

In my sophomore year, I took AP Computer Science, where I strengthened my skills in Java and built several simple games from scratch. Afterward, I ventured on my own, diving into web development with HTML, CSS, JavaScript, and PHP. One of my early projects was a form-based website. While it wasn’t perfect, it gave me a solid foundation in understanding how websites function and the basics of frontend and backend development.

During my senior year of high school, I attended the computer science program at Anthes Career Center. Spending half of each day immersed in programming, I deepened my knowledge of Python by creating a game inspired by a programming book. I also explored C++, gaining basic insights into the language, although memory management was still challenging at the time.

After high school, I enrolled at Purdue University Fort Wayne to pursue a degree in Computer Science. My first year covered familiar ground, especially in Java, so I dedicated extra time to projects like BrickThrower and various Minecraft modifications, which are showcased below. These projects allowed me to apply what I knew while sharpening my skills.

As I progressed in college, my learning expanded significantly:

  • Second Year: I delved into data structures like linked lists, stacks, queues, and binary trees. I also had my first real exposure to C programming, where I gained a deeper understanding of pointers, memory management, and how data is stored in memory.
  • Third Year: This year introduced me to database management and algorithms. I learned SQL thoroughly and explored algorithms such as sorting, searching, and recursion. My Programming Language Design course was particularly eye-opening, as it demystified how compilers work, error-checking processes, and procedural programming paradigms.
  • Senior Year: Now in my final year, I’m applying all I’ve learned to capstone projects and preparing for my career as a software engineer. From developing a Rewinder Batch Trimming & Scheduling System to exploring new technologies, I’m continually growing as both a programmer and a learner.

Through these experiences, I’ve developed a strong foundation in programming, a curiosity for exploring new technologies, and a commitment to lifelong learning. As I approach graduation, I’m excited to start my professional journey and contribute to the ever-evolving world of technology.

Projects

BrickThrower

BrickThrower is a Minecraft plugin that lets players throw bricks with a right-click, adding a fun combat and gameplay mechanic. The plugin supports Minecraft versions from 1.8 to 1.20.4, with customizable options for damage and velocity. Developed with performance and compatibility in mind, BrickThrower enhances server gameplay with a simple yet engaging feature.

TaskFlow

TaskFlow is a gamified project management app designed to help teams organize tasks, track progress, and stay motivated. Built with React, TypeScript, and Node.js, it features task management, real-time updates, and a drag-and-drop interface. Unique gamification elements, like earning points for completed tasks, enhance collaboration and engagement.

Attendance Tracker System

Attendance Tracker System is an Android-based app designed to streamline attendance management for educational institutions. Built with Java, it features QR code-based check-ins, detailed analytics, and automated email notifications for seamless administration. The app is highly customizable, supporting course and student management via an intuitive interface.

Web Crawler

Web Crawler is a system that efficiently traverses websites, extracting and organizing data into a Trie structure for rapid lookups. Developed in C with Python scripts for advanced text extraction, the crawler demonstrates effective use of algorithms and data structures for large-scale information retrieval.

CommandAlias

CommandAlias is a lightweight Minecraft BungeeCord plugin that simplifies server commands with custom aliases. Administrators can easily define alternative commands through a configuration file and reload changes instantly without restarting the server. Built for flexibility and ease of use, CommandAlias enhances server management and player accessibility.

PrefixRequest

PrefixRequest is a Minecraft plugin that streamlines custom prefix management. With an intuitive GUI, players can request personalized prefixes, and administrators can easily approve or deny them. Designed for Spigot and Paper servers, the plugin ensures efficient server moderation while enhancing player engagement.

Dragons

Dragons is a Python-based text adventure game where players engage in battles with mythical dragons, complete quests, and achieve milestones. Featuring configurable elements such as custom dragons, moves, and storylines, the game offers an immersive and customizable experience. With interactive storytelling and dynamic combat mechanics, Dragons is a demonstration of modular programming and game design.
